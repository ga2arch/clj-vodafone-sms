Send sms through italian vodafone online portal

Pre-Built Jar

Installation

Download jar

From source

Prerequisites

  • Tesseract
  • Leinigen

Installation

    git clone git@github.com:ga2arch/clj-vodafone-sms.git
    cd clj-vodafone-sms
    lein deps
    lein uberjar

Usage

    java -jar clj-vodafone-sms-1.0.0-SNAPSHOT-standalone.jar <username> <password> <phonenumber> <message>
